包含 Authorization 的完整头部，一般为 `Bearer sk-xxx`

🔄 模型映射与名单

JSON 格式不合法 以键值对表示，用于重命名外部调用的模型名称。格式为 JSON Object
JSON 格式不合法，需为数组 凡在此列表内的模型，其 Tool-calling 请求将直接透传，不注入虚拟工具。格式为 JSON Array
JSON 格式不合法，需为数组 `/v1/models` 端点主动公开呈现的模型 ID 列表。格式为 JSON Array
允许未映射模型穿透 若启用，未在映射表中的模型名称也将直接转发至上游

🛡️ 工具调用策略

解析失败时纠错重试 (FC_ERROR_RETRY) 启用后，如大模型输出工具格式损毁，代理会自动携带报错指令请求模型重写。
用于告诉大模型当前会话挂载了工具，可自行微调指令
